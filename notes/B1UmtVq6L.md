## 課程重點

### 工具分享

- [Open Frameworks](https://openframeworks.cc/)
    - C++ 為主
    - 速度快
    - 有完善的程式接口
    - 應用在大型的展覽上
    - 有完善的 3D 與 VR&AR 支援
    - 缺點：環境建設與語法複雜
- [Touch Designer](https://)
    - 圖像渲染使用 CPU 速度較快
    - 圖像化介面結合不同工具
    - VR&AR 支援
    - 費用比較高
- MaxMSP
    - 聲音處理為主
    - 用圖象介面串接與處理訊號
    - 也漸漸使用在視覺即時戶動作品上
- [Unity](https://unity.com/)
    - 遊戲渲染引擎
    - 快速處理 3D 物件與場景
- [Unreal](https://www.unrealengine.com/zh-CN/)
    - 同樣是 3D 遊戲運算引擎
    - 遊戲外也可以用在演唱會等場景
    - 圖象式介面
    - 可以在 market place 下載場景
-  [MadMapper](https://madmapper.com/)
    - 用於場景投影與視覺/聽覺訊號處理
- C4D
    - 3D 渲染軟體
    - Motion Graphics 製作
- Houdini
    - 更進階的 3D 模擬、繪圖軟體
- Three.js
    - 基於 JavaScript 的 3D 套件
    - 網頁的 3D 應用神器
- Shader 語言
- creativecoding.in
